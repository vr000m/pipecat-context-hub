# Security Policy

## Supported Versions

This project currently supports security fixes for the latest release and the
current `main` branch.

| Version | Supported |
| ------- | --------- |
| latest release | :white_check_mark: |
| `main` | :white_check_mark: |
| older versions | :x: |

## Reporting a Vulnerability

Please use GitHub's private vulnerability reporting for this repository to
report suspected security issues.

Do not open public issues or pull requests for undisclosed vulnerabilities.

When reporting a vulnerability, include:

- the affected version or commit
- a clear description of the impact
- reproduction steps or a proof of concept
- any known mitigations or suggested fixes

We will triage reports as quickly as possible and follow up through the private
reporting channel.
